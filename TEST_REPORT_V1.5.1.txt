# Отчёт проверки v1.5.1 — Обновление приложения

База: `-trenirovki-nikity-main 12.zip`.

Проверено:

1. Синтаксис JavaScript:
   - `app.js` — успешно;
   - `db.js` — успешно;
   - `seed.js` — успешно;
   - `exercise-guides.js` — успешно;
   - `version.js` — успешно;
   - `service-worker.js` — успешно.

2. Manifest:
   - `manifest.webmanifest` валиден как JSON.

3. Service Worker:
   - `SERVICE_WORKER_RELEASE` = 1.5.1;
   - `version.js` = 1.5.1;
   - имя кэша обновлено до `nikita-workouts-v1.5.1-update-manager`;
   - все файлы из `APP_SHELL` существуют;
   - установка использует `cache: reload`;
   - сетевые запросы оболочки при отсутствии кэша используют `cache: reload`;
   - добавлены сообщения `SKIP_WAITING` и `CLEAR_APP_CACHES`.

4. IndexedDB:
   - версия базы осталась v3;
   - новые хранилища не добавлялись;
   - существующие хранилища не удалялись;
   - миграции нет.

5. Логика обновления проверена статически:
   - добавлена проверка `version.js` с cache-busting;
   - добавлен баннер доступного обновления;
   - добавлены кнопки «Проверить обновление» и «Обновить кэш» в разделе «Ещё»;
   - очистка затрагивает только Cache Storage с именами `nikita-workouts-*`;
   - IndexedDB не очищается;
   - черновик тренировки сохраняется перед перезагрузкой.

6. ZIP:
   - архив собран;
   - целостность архива проверена через `unzip -t`.

Ограничение:

Интерактивный браузерный прогон в sandbox не выполнялся. Проверка выполнена статически и через синтаксический контроль.
