# v1.6.0 — Нагрузка мышц

Большое функциональное обновление после v1.5.2.

Что добавлено:

- Новый раздел: Прогресс → Мышцы.
- Переключатель периода: 7 дней / 14 дней.
- Расчёт рабочих подходов по группам:
  - грудь;
  - спина;
  - плечи;
  - бицепс;
  - трицепс;
  - ноги;
  - ягодицы;
  - пресс.
- Статусы по каждой группе:
  - мало;
  - нормально;
  - много;
  - перегруз.
- Короткий вывод: где недобор, где много нагрузки, где лучше не добавлять вес/подходы.
- На главной добавлена карточка «Мышцы за 7 дней».
- Упражнения-комбо засчитываются в несколько групп: например жим — грудь/трицепс, тяга — спина/бицепс.
- Для пользовательских упражнений приложение пытается определить группу по названию и полю «группа».

Данные:

- IndexedDB остаётся v3.
- Новых хранилищ нет.
- Миграции базы нет.
- Профили, история, фото, боль, рекорды, программы и черновики не очищаются.
- Нагрузка мышц считается из уже сохранённой истории текущего профиля.

Service Worker:

- Новый кэш: nikita-workouts-v1.6.0-muscle-load
