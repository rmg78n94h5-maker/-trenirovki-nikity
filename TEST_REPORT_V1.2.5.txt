# Отчёт проверки v1.2.5

## Проверено статически

- `node --check` для всех JavaScript-файлов — успешно.
- `manifest.webmanifest` проходит JSON-проверку.
- Все файлы из `APP_SHELL` Service Worker существуют.
- Версия `version.js`: 1.2.5.
- Версия `service-worker.js`: 1.2.5.
- Имя кэша: `nikita-workouts-v1.2.5-program-builder`.
- IndexedDB оставлена на версии 2.

## Проверено по коду

- В разделе «План» используется список уникальных программ, а не прямой вывод всех записей из IndexedDB.
- Стандартная программа и её личная копия с тем же названием отображаются как один пункт.
- Намеренно созданные копии с изменённым названием остаются отдельными программами.
- Переключение на стандартный шаблон не плодит новые копии, если личная копия уже есть.
- Кнопка «＋ Создать» открывает конструктор программы.
- Конструктор создаёт 1–7 пустых тренировочных дней и открывает новую программу.
- Редактор дня остаётся прежним: через ✎ можно добавлять упражнения, менять подходы, повторы, вес и отдых.

## Ограничения проверки

Полноценный ручной прогон на реальном iPhone невозможен внутри sandbox. Перед загрузкой на GitHub рекомендуется сделать экспорт резервной копии в приложении.
